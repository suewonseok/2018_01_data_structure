RelationType PaperType::CompareByName ( const PaperType &  data)

record의 Name를 토대로 두 개의 PaperType을 비교한다.

두 개의 PaperType을 비교한다.

Precondition
두 개의 PaperType이 초기화된 상태여야한다.
Postcondition
새로운 학술대회 정보에 지정 파일의 데이터가 포함된다.
Parameters
datatarget Name for comparing.
Returns
만약 this.Name < data.Name 이면 LESS를 반환, 만약 this.Name > data.Name 이면 GREATER을 반환, 그렇지 않으면 EQUAL를 반환.
